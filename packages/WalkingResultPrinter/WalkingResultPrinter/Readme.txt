=============
活動量計アプリケーション補助ツール
=============
活動量計アプリケーションでデータを取り込んだ後の、集計ファイル作成・グラフ作成・グラフ印刷を自動的に行います。

■必要なソフトウェア
.NET Framework 4.5.2
Microsoft Excel

■使い方
1. 活動量計アプリケーションでデータ取り込みを行った後、活動量計アプリケーションのメニュー画面を表示した状態で、「全被験者の集計ファイル作成」ボタンを押し、集計ファイルを作成します。
2. 活動量計アプリケーションのフォルダにある"UserData"フォルダを、このツールのフォルダにコピーします。
3. グラフの期間を「1週間分」または「2週間分」から選び、「グラフ印刷」ボタンを押して、手順1で作成した各集計ファイルのグラフを印刷します。
   また、"sheets"フォルダにグラフのエクセルシートが出力されます。
   なお、「グラフのエクセルシート作成」ボタンを押すと、印刷は行わず、エクセルシートのみ作成します。

■注意
・印刷プレビューやプリンタの設定は行えません。印刷の設定を行いたい場合は、生成されたエクセルシートを直接開いて印刷してください。
・グラフの印刷やエクセルシート作成中にエラーが発生した場合、エラーの内容を"error.txt"というファイルに出力し、次の集計ファイルのグラフに移ります。
・生成されるグラフシートファイルを開いたまま実行すると、新規グラフ生成時にエラーが発生する場合があります。グラフ生成・印刷は、ファイルを閉じてから行ってください。
・エラーが発生するときは、一度コンピューターを再起動して再度本プログラムを実行すると成功する場合があります。
・手動で印刷する場合は、"sheets"フォルダに出力されたエクセルシートの「週サマリ1(2)週間」「日METs1(2)週間」シートを印刷してください。
